# Vue.js based Ethereum Blockchain viewer

## Project setup
```
npm install
```
Copy `.env.example` to `.env`, then add your Infura key.


### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
